# Free PDF Tools

![Free PDF Tools Preview](preview.png)


A collection of free online PDF tools that work directly in the browser.
No software installation, no signup, and no file upload required.

## Available Tools

- Split PDF pages online
- Merge multiple PDF files
- Compress PDF files
- Convert JPG to PDF
- Convert PDF to JPG

All tools are designed to be fast, simple, and privacy-friendly.

## Live Demo

You can try the tools here:
https://free-pdf-tools-e7x.pages.dev

## Why use browser-based PDF tools?

- No file upload to servers
- Works on Windows, macOS, and mobile
- Unlimited usage
- No watermarks
- Ideal for students and professionals

## Technology

- Client-side JavaScript
- PDF processing in browser
- Modern HTML, CSS

---

This project demonstrates how PDF tools can be built and used safely without storing user files.
